# 🏡 Casa del Niño - Landing Page

> _Un espacio de amor, contención y crecimiento para los niños._

Bienvenido al repositorio oficial de la landing page de **Casa del Niño**. Este proyecto busca conectar a la comunidad con nuestra misión, facilitando información sobre quiénes somos, qué hacemos y cómo pueden colaborar.

---

## ✨ Características

Este proyecto ha sido diseñado con un enfoque en la **estética moderna**, la **accesibilidad** y una **experiencia de usuario fluida**.

- 🎨 **Diseño Moderno & Responsivo**: Adaptable a cualquier dispositivo (móvil, tablet, desktop).
- ⚡ **Alto Rendimiento**: Construido con [Vite](https://vitejs.dev/) para una carga instantánea.
- 🎭 **Animaciones Suaves**: Integración de [Framer Motion](https://www.framer.com/motion/) para dar vida a la interfaz.
- 💅 **Estilizado con Tailwind**: Uso de [Tailwind CSS](https://tailwindcss.com/) para un diseño limpio y consistente.
- ♿ **Accesible**: Pensado para ser utilizable por todos.

## 🚀 Comenzando

Sigue estos pasos para ejecutar el proyecto localmente:

1.  **Clonar el repositorio**

    ```bash
    git clone https://github.com/tu-usuario/casa-del-nino.git
    cd casa-del-nino
    ```

2.  **Instalar dependencias**

    ```bash
    npm install
    ```

3.  **Iniciar el servidor de desarrollo**

    ```bash
    npm run dev
    ```

4.  ¡Abre tu navegador en `http://localhost:5173` y disfruta!
